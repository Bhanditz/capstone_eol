8     # Builds a DataObject and creates all of the ancillary relationships.  Returns the DataObject.
 
9     #
 
10     # As a matter of course, you had better load the foundation scenario before using this.  It relies on many of the
 
11     # enum-like classes being populated.
 
12     #
 
13     # This takes three arguments.  The first is the type of the DataObject: acceptable values include 'Text',
 
14     # 'Image', 'GBIF Image', 'Flash', and 'YouTube'.  The second argument is the description, since all
 
15     # DataObject instances make use of this one.  The third argument is a hash of options.  Some key values are:
 
16     #
 
17     #   +agent+::
 
18     #     If specified, this dato will be related to this agent.  If you don't supply this, an agent will try to be
 
19     #     intelligently deduced from other options, or picked as the last Agent created, or created from scratch if
 
20     #     nothing else worked.
 
21     #   +agent_role+::
 
22     #     If specified (as either a string or an AgentRole instance), this dato will be related to the agent with
 
23     #     this role.  If missing, 'Author' will be used.
 
24     #   +content_partner+::
 
25     #     If specified, a Resource, HarvestEvent, and all relationships between them will be created.
 
26     #   +hierarchy_entry+::
 
27     #     Which HE to link this object to. If missing, defaults to the last HE in the table, so be careful.
 
28     #   +language+::
 
29     #     Which language you want this to be given.  If missing, defaults to English.
 
30     #   +license+::
 
31     #     Which license you want this to be given.  If missing, defaults to the first license in the DB.
 
32     #   +mime_type+::
 
33     #     If you care about it, set it.  Otherwise, a useful default will be chosen.
 
34     #   +name+::
 
35     #     Which Name object (q.v.) to link the Taxon to. 
36     #   +num_comments+::
 
37     #     How many comments to attach to this object.
 
38     #   +scientific_name+::
 
39     #     Which raw scientific name the content provider assigned to the Taxon.  Defaults to the HE's. 
40     #   +taxon+::
 
41     #     Which Taxon (q.v.) to link this to.  If missing, one will be created. 
42     #   +toc_item+::
 
43     #     If this is a 'Text' type, this specifies which TocItem to link this to. If left blank, one is randomly
 
44     #     chosen.
 
45     #
 
46     # Any other options on this method will be passed directly to the DataObject#gen method (for example,
 
47     # lattitude, longitude, and the like).
 
48     #
 
49     # NOTE - I am not setting the mime type yet.  We never use it.  NOTE - There are no models for all the
 
50     # refs_* tables, so I'm ignoring them.
